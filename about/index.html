Why this blog?

I like JavaScript because it is very easy to get things done, fast. The language itself has some quirks, but once you know what features to use and what not to use, then you’ll get productive and can write robust code.

With the advent of NodeJS, server-side JS development has skyrocketed. Many big companies are using it, and the node package repository (npm) has over 100.000 modules available to your disposal. Like lego-blocks, you can assemble them to fit your needs, and if there is no module that fits your needs, you write it yourself. No reasons to re-invent the wheel all over again.

I want to explore things that interest me, that I would love to learn more about, and by writing about it you learn it better. I want to become a better developer, expand my horizons, learn best practices and share cool stuff with others.
